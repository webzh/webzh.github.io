Nginx-geoip2模块-限制国家地区访问

发表于 2019-03-14 | 分类于 服务器
下载GeoIP数据库

首先需要去Maxmind官网下载最新的GeoIP地区数据库
MaxMind旗下的GeoIP2服务能识别互联网用户的地点位置与其他特征，应用广泛，包括个性化定制内容、诈欺检测、广告定向、网站流量分析、执行规定、地理目标定位、地理围栏定位 (geo-fencing)以及数字版权管理。MaxMind的GeoIP2数据库为大容量环境提供IP智能数据（官方介绍）。通俗来说，MaxMind的GeoIP2数据库提供准确的IP信息，包括IP地址的位置（国家、城市、经纬度）。现在其GeoIP已经升级到第二代，也提供了免费版本的IP数据库（GeoLite2），对于识别IP地址来源国家比较准确，要识别到城市，需要付费购买精准版。

GeoLite2 开源数据库下载地址

安装libmaxminddb依赖库

libmaxminddb依赖库，主要用于读取MaxMind DB文件，包括MaxMind下的GeoIP2数据文件。

安装ngx_http_geoip2_module编译Nginx

重新编译ngx_http_geoip2_module模块。原Nginx带的–with-http_geoip_module用的是使用的是第一代Geoip（geoip1）
git clone –recursive https://github.com/leev/ngx_http_geoip2_module

增加模块
–add-module=/usr/local/src/ngx_http_geoip2_module \

make

#备份原nginx程序文件

#mv /usr/local/nginx/sbin/nginx /usr/local/nginx/sbin/nginx.bak &&
\cp objs/nginx /opt/webserver/nginx/sbin/nginx &&
nginx -t

#测试没有问题，升级nginx
make upgrade
